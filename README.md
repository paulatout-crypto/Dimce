DimCE 

1. Prerequisites for Deployment 

Verify that MATLAB Runtime(R2024b) is installed.   
If not, you can download the MATLAB Runtime installer.
at this location:
  
   https://kdrive.infomaniak.com/app/share/1391365/c89fa612-4f24-4d78-b7da-c4f7051680b3 
	
NOTE: You will need administrator rights to run the MATLAB Runtime installer. 

Alternatively, download and install the Windows version of the MATLAB Runtime for R2024b 
from the following link on the MathWorks website:

    https://www.mathworks.com/products/compiler/mcr/index.html
   
For more information about the MATLAB Runtime and the MATLAB Runtime installer, see 
"Distribute Applications" in the MATLAB Compiler documentation  
in the MathWorks Documentation Center.

1. Conditions préalables au déploiement

Vérifiez que MATLAB Runtime (R2024b) est installé.
Si ce n'est pas le cas, vous pouvez télécharger le programme d'installation de MATLAB Runtime.
à cet emplacement :

https://kdrive.infomaniak.com/app/share/1391365/c89fa612-4f24-4d78-b7da-c4f7051680b3

REMARQUE : vous aurez besoin des droits d'administrateur pour exécuter le programme d'installation de MATLAB Runtime.

Vous pouvez également télécharger et installer la version Windows de MATLAB Runtime pour R2024b
à partir du lien suivant sur le site Web de MathWorks :

https://www.mathworks.com/products/compiler/mcr/index.html

Pour plus d'informations sur MATLAB Runtime et le programme d'installation de MATLAB Runtime, consultez
"Distribuer des applications" dans la documentation du compilateur MATLAB
dans le centre de documentation MathWorks.
